Microsoft Word 2007

Introduction

Microsoft Word is a word processing software developed by Microsoft. It was first released on October 25, 1983, under the name Multi-Tool Word for Xenix systems.

Release date: October 1983
Programming languages: C++, Objective-C
Original author: Microsoft Corporation, Richard Brodie
Platforms: x86-32 (32 bit Intel x86), x86-64, ARM architecture family, AArch64

Opening the MS word

  1. Start button >> All Apps >> Microsoft Office >> Microsoft Word 2007
  2. Start button >> All Programs >> Microsoft Office >> Microsoft Word 2007
  3. Run Command (Win+R) >>Winword>> Microsoft Word 2007

Microsoft Office Word 2007 Button

New:-This option can get a blank or New Document.
Open:-This option allows to user open an existing document or file.
Save:-This option is used to save the current document file.
Save As:-Save As option allow to user to save a file with a different name or create a duplicate file
Print:-This option allows you to print the active document. Print command displays more options such as print, Quick Print and Print Preview.
Prepare:-This command allow you to monitor properties of the document, inspecting document, Encrypting Document, restricting permission, adding digital signature, making read only and run compatibility checker
Send:-This tool allows you to send your document somewhere via Email and sending the active document to internet fax.
Publish:-This allows you to publish the document to blog, creating document management server and document workspace.
Close:-This option closes the active document. Or press shortcut Ctrl+W, Alt+F4 (Close the program)

Text Selection with Keybaord Keys

  • Shift+rightarrow- select a character in right side
  • Shift+leftarrow- Select a character in left side
  • Shift+downarrow- Select a line in the down directionSelect a line in the down direction
  • Shift+uparrow- Select a line in the up direction
  • Shift+Home- Select a line from left to right
  • Shift+End- Select a line form right to left
  • Ctrl+Shift+rightarrow- Select a word in the right side
  • Ctrl+Shift+leftarrow- Select a word in the left side
  • Ctrl+Shift+downarrow- Select a paragraph in the down direction
  • Ctrl+Shift+uparrow- Select a paragraph in the up direction

Mouse Functions

  • Single click to beginning of the line-Select a line
  • Double click to beginning of the line-Select a paragraph
  • Triple Click to beginning of the line-All Select
  • Double click to the word-Select a word
  • Ctrl+Click to beginning of the line-Select Optional lines
  • Shift+Click to beginning of the line-Select Multiple lines

Cut, Copy & Paste

  1. Home tab >> clipboard group >> Cut, copy, paste and format painter
  2. Ctrl+X or Shift+Delete-Cut the selected text
  3. Ctrl+V or Shift+Insert-Paste the copy text
  4. Ctrl+C or Ctrl+Insert-Copy the selected text

Paste Special

The paste special option is used to paste matter to any desired position within same document or in other document or in other software. The paste special can be used to paste the copy data with or without formatting into the document.
Or press shortcut Ctrl+Alt+V

Go to command or tool

Go to tool is used to go to specified page, line, table, section etc. You use this tool in the find and replace dialog box in the editing group of the Home tab. Or press shortcut Ctrl+G or F5.

Page Setting Up

You can do page setting of your document by that you can be able to set paper size, Page Margin (top, bottom, Left & Right and Gutter), Page Orientation, Page Layout etc.
You can set the page setting by using following steps-

  1. Page Layout:-This tool contains setting to select section start, Header & Footer and Borders.
  2. Orientation:-This tool sets the orientation of the page such as portrait & Landscape.
  3. Go to Page Layout Tab in MS word 2007 & 2010.
  4. Click the page setup dialog box launcher button to open the page setup dialog box.
  5. Set the page size, Margins, orientation, columns etc. and click ok.

Print through print dialog box

You can print the document page even through print dialog box directly to press the shortcut key Ctrl+P or Ctrl+Shift+F12.
The print dialog box will appear with the more options such as printer, print range, selecting what to print and print options.

Character and paragraph formatting

Microsoft word is made with more features for formatting the paragraph and characters like as cut, copy, paste,bullets & numbering, font face, font size, background colour, fontcolour, line spacing, indenting etc.

Font face& Font Size

Fonts are the face of the text appearance entered in the document. We can change the font and font size of the selected text in the document.

  1. Home tab >> Font Dialog box Launcher button >>fontsize and font face
  2. Press the shortcut key Ctrl+D or Ctrl+Shift+F or Ctrl+Shift+P.
